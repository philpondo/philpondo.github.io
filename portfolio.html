GoPlay

GoPlay is a web app that allows VALORANT players to find teammates of a similar skill set and play style. The app is built using Ruby on Rails as a backend framework, and Vue.js as a frontend framework. The app enables users to create a post outlining what kind of teammates they are looking for, so other users can respond to their posts with a direct message to team up.
